# Backend

This is the Express + TypeScript backend skeleton for Tourbnb.

## Setup
Copy `.env.example` to `.env` and update with your database URL and secrets.

```powershell
npm ci
npm run dev
```

## Scripts
- `npm run dev` — start dev server via nodemon & ts-node
- `npm run build` — compile TypeScript
- `npm start` — start Node from `dist`
