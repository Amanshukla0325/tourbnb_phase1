## Form Input Responsive Behavior

### All Screen Sizes:
- **Height**: Consistent h-12 (48px)
- **Padding**: px-4 consistent
- **Border Radius**: rounded-xl (12px) consistent
- **Focus State**: Focus ring appears same size

### On Mobile:
- **Width**: Full width of container (max-w-md on right section)
- **Container Padding**: p-6 sm:p-8

### On Desktop:
- **Width**: Full width of right section (50% of screen)
- **Container Padding**: p-12

---

## Color Responsive System

### Light Mode (Default)
```
Background: Gradient to-white / from-slate-50
Input BG: slate-100
Focus Ring: Primary color with transparency
Text: gray-900 for headings
```

### Dark Mode (If Implemented)
```
Would use CSS custom properties:
--background: dark
--foreground: light
--primary: adjusted for dark
--border: lighter gray
```

---

## Typography Responsive Sizing

### Headings:
- **Mobile**: text-3xl (30px)
- **Tablet**: text-3xl sm:text-4xl
- **Desktop**: text-5xl (admin login left section)

### Body Text:
- **Mobile**: text-base (16px)
- **Tablet**: text-base sm:text-lg
- **Desktop**: text-lg (20px) for descriptions

### Labels:
- **All**: text-sm (14px) consistent

---

## Icon Sizing

### Login Icons:
- **Form Header Icon**: w-7 h-7 on mobile, w-8 h-8 on desktop
- **Branding Section Icon**: w-8 h-8 consistent
- **Social Buttons**: w-5 h-5 consistent

### Dashboard Icons:
- **Stat Card Icons**: w-8 h-8 consistent
- **Action Icons**: w-5 h-5 for buttons
- **Chart Icons**: Rendered by recharts

---

## Button Responsive Behavior

### Height:
- **Login Buttons**: h-12 (mobile/desktop)
- **Social Buttons**: h-11 (mobile/desktop)
- **Dashboard Buttons**: h-10 to h-12 depending on context

### Width:
- **Primary Buttons**: w-full in forms
- **Social Buttons**: Grid-based (2 columns)
- **Action Buttons**: Inline in tables

### Padding:
- **Login Buttons**: px-4 (managed by Input component)
- **Dashboard Buttons**: px-4 to px-6
- **Consistent**: py managed by height

---

## Shadow & Depth Responsive

### Shadow Levels:
- **Cards**: shadow-lg consistently
- **Hover States**: shadow-xl on desktop
- **Backdrop**: backdrop-blur-xl consistent

### Z-index Stack:
- **Background Blurs**: -z-10
- **Content**: z-10
- **Modals**: z-50 (future)
